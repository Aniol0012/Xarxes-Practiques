### Descripció:

Repositori de la pràctica 1 de Xarxes, Programació d'aplicacións de xarxa.

### Instal·lació:

   ```sh
   git clone https://github.com/Aniol0012/Xarxes-Practica-1
   ```
### Ús:
1. Compilar el client:
    
    ```sh
    make
    ```
    
2. Obrir el servidor:
    ```sh
    ./server.py
    ```

3. Obrir el client/clients en un altre terminal:
    ```sh
    ./client
    ```
